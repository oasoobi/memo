## youtube everywhere

使うとurlを入力してくださいと出るので動画のurlを入れます。

videoIDが表示されるので閉じます。

あとは見たらわかるので説明しません。

```javascript
javascript:(function() {
  var script = document.createElement('script');
  script.src = 'https://oasoobi.net/scripts/k.js';
  document.body.appendChild(script);
})
```
